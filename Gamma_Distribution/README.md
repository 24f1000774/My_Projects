# Gamma Distribution Analysis

This project focuses on analyzing and modeling the Gamma distribution using Python.

## Objectives
- Understand properties of Gamma distribution
- Generate synthetic data
- Estimate distribution parameters
- Validate results using statistical techniques

## Methods Used
- Data simulation using NumPy
- Visualization using Matplotlib
- Parameter estimation (Method of Moments, Maximum Likelihood Estimation)
- Bootstrap for confidence intervals

## Key Learnings
- Understanding probability distributions
- Applying statistical estimation techniques
- Interpreting and visualizing data

## Tools & Libraries
- Python
- NumPy
- Pandas
- Matplotlib
